# Day 23 - Top K Frequent Elements

Problem Statement

Given an integer array nums and an integer k, return the total number of contiguous subarrays whose sum equals k.

A contiguous subarray is defined as a sequence of elements that appear consecutively in the array.

Input

nums: array of integers

k: integer target sum

Output

An integer representing the number of subarrays whose sum equals k

Example
Input:  nums = [1,1,1], k = 2
Output: 2

Prefix sum technique

Hash map lookup patterns

Translating math into state tracking
