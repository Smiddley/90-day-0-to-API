Problem Statement

Given two sorted arrays nums1 and nums2, return a new array containing all elements in sorted order.

Input

nums1: sorted array

nums2: sorted array

Output

A new sorted array

Example
Input:  [1,3,5], [2,4,6]
Output: [1,2,3,4,5,6]

Constraints

Do not re-sort the final array

Time complexity: O(n + m)

Concept

Two pointers

Linear merge

Cloud Analogy

Merging two sorted log streams by timestamp.
