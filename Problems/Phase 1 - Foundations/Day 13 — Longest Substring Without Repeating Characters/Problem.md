Problem Statement

Given a string s, find the length of the longest substring without repeating characters.

Input

s: string

Output

Integer (max length)

Example
Input:  "abcabcbb"
Output: 3
Explanation: "abc"

Constraints

Time complexity: O(n)

Concept

Sliding window

Hash set / map

Cloud Analogy

Longest uninterrupted session without credential reuse.
